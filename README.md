# Roman Number Calculator

### prepare

```sh
    $ npm install -g @ionic/cli
    $ npm install
```

### start dev

```sh
    $ ionic serve --ssl
```

### start unit test

```sh
    $ ng test
```

### start e2e test

```sh
    $ npx cypress open
```

### build

```sh
    $ ionic build
```

### sync to ios

```sh
    $ npx cap sync
```

### open xcode

```sh
    $ npx cap open ios
```

### open androidstudio

```sh
    $ npx cap open android
```

### code style

```sh
    $ ng lint
```
