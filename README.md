Roman Number Calculator
=======================

### prepare
```sh
    $ npm install
```

### start dev
```sh
    $ ng serve -o
```

### start unit test
```sh
    $ ng test
```

### start e2e test
```sh
    $ npx cypress open
```

### build
```sh
    $ ionic build
```

### sync to ios
```sh
    $ npx cap sync
```

### open xcode
```sh
    $ npx cap open ios
```
